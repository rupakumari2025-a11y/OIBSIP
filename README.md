# 👗 VogueVista - Fashion Brand Landing Page

## 📌 Project Overview

VogueVista is a modern and responsive fashion brand landing page developed using **HTML5** and **CSS3** as part of the **Oasis Infobyte Web Development & Designing Internship (Level 1 - Task 1)**.

The website features a clean and attractive design with a responsive layout that provides a great user experience across different screen sizes.

---

## 🚀 Features

- ✅ Responsive Navigation Bar
- ✅ Hero Section with Call-to-Action Button
- ✅ Featured Collection Section
- ✅ About Us Section
- ✅ Contact/Footer Section
- ✅ Responsive Design using CSS Flexbox
- ✅ Modern Typography and Color Palette
- ✅ Hover Effects on Buttons and Cards

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Web Design
